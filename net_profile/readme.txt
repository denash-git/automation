net profil - переключает режим профиля Общественная сеть ---> Сеть предприятия
			   через реестр, актуально на ОС Windows 7
				
Работа основывается на параметре реестра DateLastConnected, по которому определяется текущий профиль
 и произодиться замена ключа, если он не соответствует.

В модуле ****** заменить -u login -p pass c правами локального администратора на удаленном ПК.

коды возврата %errorlevel% :

			0 - изменение не требуется
			1 - скрипт выполнил изменение
			2 - целевая ОС не поддерживается
