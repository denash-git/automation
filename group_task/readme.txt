group_task - менеджер группового выполнения задач (модулей) :modul. Обеспечивает перебор по заданному 
		листу ip-адресов и удаленное выполнение модуля на каждом ПК, с отражением результата.
		
			
	из файла list.csv построчно читается строка ip, flag пинга, flag задачи, comment :
		- ip целевой ip host`a
		- flag_task =0 -требуется выполнение :modul. не равно 0 -задача выполнена, не подлежит выполнению
		- flag_ping история доступности hosta, 0 или 1. Модуль выполняется только при доступности hosta
				в текущее время. Флаг хранит историческое значение.
		- comment для информации, результат
			 
		Входящий параметр модуля ip хоста. Выходной парметр результат задачи flag_task и comment
